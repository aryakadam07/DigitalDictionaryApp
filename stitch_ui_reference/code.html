Ephemeral

adjective /əˈfem(ə)rəl/

Lasting for a very short time.

Example Usage

"Fashions are ephemeral, but style is eternal."

Synonyms
transitory fleeting momentary
Antonyms
permanent eternal

history_edu Etymology

Late 16th century: from medieval Latin ephemeralis, from Greek ephēmeros ‘lasting only a day’.
